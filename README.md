# CAREJA-UM
Careja adalah platform berbasis web yang membantu mahasiswa dan fresh graduate membuktikan kemampuan melalui project nyata. Pengguna dapat mengerjakan proyek dari perusahaan dan menjadikannya portofolio dinamis. Careja menjembatani kebutuhan pengalaman kerja dengan peluang, sehingga proses rekrutmen lebih objektif dan berbasis hasil kerja.
